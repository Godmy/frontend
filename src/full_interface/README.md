# Full Interface Components

This directory contains all specific functionality that was extracted from the main frontend package.

## Structure
```
src/
└── full_interface/              # All specific functionality moved here
    ├── features/                # Business features
    ├── pages/                   # Specific pages
    ├── processes/               # Business processes
    ├── shared/                  # Shared components
    ├── widgets/                 # UI widgets
    ├── user_dashboard/          # User-specific components
    └── admin_dashboard/         # Admin-specific components
```

These components are not part of the submodule but are included in the full package for internal use.
