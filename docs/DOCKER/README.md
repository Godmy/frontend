# Docker Documentation

This directory contains comprehensive documentation for Docker usage in the Multipult Frontend project.

## Overview

Docker provides containerization for consistent development, testing, and production environments. Our setup includes:

- Multi-stage builds for optimized production images
- Development mode with hot-reload
- Docker Compose for orchestrating multi-container applications
- Environment-specific configurations
