# Документация для папки `src/entities/concept`

## Описание
Папка `src/entities/concept` содержит бизнес-сущность "Концепция" - основную сущность, представляющую понятия в онтологии.

## Содержимое
Директория содержит:

- Типы данных для концепций
- Компоненты отображения информации о концепциях
- Утилиты для работы с концепциями
- GraphQL-запросы и мутации для взаимодействия с API
